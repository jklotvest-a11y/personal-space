# 个人作品展示网站（可直接部署 Vercel）

这是一个带设计感的单页个人网站模板，适合展示你的作品、能力与联系方式。  
项目采用纯静态方案（`HTML + CSS + JavaScript`），无需构建工具，部署简单、加载快、维护成本低。

---

## 1. 当前已完成内容

- 有视觉风格的深色渐变页面（Hero + 作品 + 关于 + 联系）
- 响应式布局（桌面 / 平板 / 手机）
- 手机端折叠菜单
- 滚动入场动效
- 自动显示当前年份
- Vercel 部署配置（`vercel.json`）

---


## 3. 如何本地查看

你可以任选一种方式：

### 方式 A：直接双击

直接双击 `index.html`，浏览器即可打开。

### 方式 B：本地静态服务（推荐）

如果你有 Node.js，可运行：

```bash
npx serve .
```

然后访问命令行提示的地址（通常是 `http://localhost:3000`）。

## 4. 如何改成你的个人信息

### 4.1 修改基本信息

在 `index.html` 中搜索并替换：

- `YOUR NAME` -> 你的名字或品牌名
- `yourname@email.com` -> 你的邮箱
- GitHub / Behance 链接 -> 你的真实链接

### 4.2 修改作品内容

在 `index.html` 的“精选作品”区域中，每个 `<article class="card">` 是一个项目卡片：

- `tag`：项目类型（品牌官网 / 活动页 / 数据可视化）
- `h3`：项目名称
- `p`：项目成果描述（建议写数据结果）
- `a href`：项目详情链接（可以是 Notion、掘金、博客、线上地址）

### 4.3 修改主视觉文案

`hero` 区块中的：

- 主标题 `h1`
- 自我介绍 `hero-desc`
- 按钮文案

都可以替换成你的风格。

## 5. 设计说明（为什么这样设计）

- 深色渐变背景 + 玻璃感导航：更现代，也更突出作品卡片
- 卡片轻微悬浮交互：增强“设计感”，但不喧宾夺主
- 简洁信息层级：让访问者快速看懂你是谁、做过什么、如何联系
- 响应式优先：在手机上也保持信息清晰和可点击性

## 6. 部署到 Vercel（一步步）

## 方式 A：通过 GitHub（推荐）

1. 把当前项目上传到 GitHub 仓库
2. 打开 [Vercel](https://vercel.com/) 并登录
3. 点击 **Add New... -> Project**
4. 选择你的 GitHub 仓库并导入
5. Framework Preset 选 **Other**（静态站）
6. 直接点击 **Deploy**
7. 部署成功后会得到一个 `*.vercel.app` 域名

### 方式 B：使用 Vercel CLI

```bash
npm i -g vercel
vercel
```

按提示选择当前目录并确认部署即可。
